How to Install the Mini Reloader Extension

Thank you for choosing the Mini Reloader Extension! Follow these steps to install the extension:

  1. Open Google Chrome.
  2. Navigate to chrome://extensions/.
  3. Enable "Developer mode" by toggling the switch in the top right corner.
  4. Click on "Load unpacked".
  5. Select the folder where the extension was extracted (e.g., %cd%\extension) and click "Select Folder".
  6. The extension should now be installed and ready to use.

If you encounter any issues, please refer to the documentation or contact support.
